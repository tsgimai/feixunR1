# feixunR1
斐讯R1音响软件，调试教程

1、音响开启连接模式，手机安装配网.apk，然后按照操作，让音箱连上你的 Wi-Fi，记下它的 IP；
2、ip地址安装后，adb打开安装教程；
3、直接安装（推荐）把 APK （安装给音响的，不是手机）放到你电脑当前目录
4、精准（PowerShell）wifi自动连接
注：想恢复小讯，只要按住头部通电变紫灯恢复出厂设置就行了。
